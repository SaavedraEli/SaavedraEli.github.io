Información de la Página

Una página web es un documento accesible desde cualquier navegador con acceso a internet, y que puede incluir audio, vídeo, texto y sus diferentes combinaciones. Las primeras webs surgieron en el año 1992. Se trataban de páginas estáticas, y que sólo podían ser accesibles para ser modificadas a través de código html.
